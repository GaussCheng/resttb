resttb
======

This tool is use to generate a table from a plant text.If you have some document just like 

h1:h2:h3:h4
1: col1  : col2 : col3
2: col11 : col22 : col33

you can use this tool to generate a table use for restructText , just like
+-----+---------+-------+-------+
| 1   | col1    | col2  | col3  |
| 2   | col11   | col22 | col33 |
+-----+---------+-------+-------+
